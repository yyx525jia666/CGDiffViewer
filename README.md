# 漫画处理器 (v1.1)

当我们在观看CG差分时，很多差分一大堆图片都是一样的，那么我一页一页翻就很不爽，但是制作成GIF又不方便不好用，所以就有了这个东西。本质自用，之后想到的能使用的功能还会加。
一补，增加了图片删除（当我们下载下来漫画时，一些无良汉化组会加入一大堆广告，可以自动删除并重命名，备份和撤销）和文件夹合并（整合本，合集等等，支持调整顺序和自动重命名，压缩）的功能。

删除方面，双击进入全屏页面，在多选页面长按可以滑动多选

## 新版功能 (v1.1)

1. **图片删除板块**：
    - 支持在预览模式下快速勾选删除图片。
    - **自动重命名**：删除图片后，文件夹内的其余图片会自动按顺序（1, 2, 3...）重新命名，保持序列完整。
    - **撤回机制**：支持撤回误删操作，删除的图片会暂时保存在隐藏目录中。
    - **进度显示**：处理大量图片时提供进度条反馈。

2. **文件夹合并板块**：
    - **项目化管理**：可以创建多个合并项目，保存合并记录。
    - **多文件夹顺序合并**：将多个文件夹中的图片按顺序合并到一个目标文件夹中。
    - **首图预览**：自动提取源文件夹第一张图片作为封面。
    - **导出ZIP**：一键将合并后的成果打包成压缩包，修复了递归压缩导致的体积异常问题。

## 核心播放功能

1. **图片序列播放**：按文件名顺序播放图片序列。
2. **精确速度控制**：播放速度精确到小数点后一位（如20.5张/秒）。
3. **全屏横屏播放**：适配不同比例的CG。
4. **长按加速**：长按屏幕可触发设定的倍速播放。
5. **断点续播**：自动记录每个序列的上次观看位置。

## 项目结构

```
app/src/main/java/com/example/imageviewer/
├── MainActivity.kt          # 主入口
├── MainScreen.kt            # CG观看历史列表
├── ui/
│   ├── deletion/            # 图片删除板块
│   ├── merge/               # 文件夹合并板块
│   ├── theme/               # 主题配置
│   └── components/          # 通用组件
├── data/                    # 数据库与实体类
└── util/                    # 文件、ZIP处理工具类
```

## 构建与运行

1. 使用 Android Studio 打开项目。
2. 确保 Gradle 已同步。
3. 连接 Android 设备，点击运行。
4. **权限说明**：应用需要“所有文件访问权限”以执行删除和重命名操作。
